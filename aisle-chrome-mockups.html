Notes
A — Featherweight: Strongest suppression. 11px / medium / gray-400. Reduces button padding from 16px → 10px. Chevron shrinks to 14px.
B — Title case: Drops ALL-CAPS and letter-spacing. Same gray-400, 13px, 500-weight. Feels like a nav breadcrumb rather than a section heading.
C — Hairline rule: Turns the header into a section-divider motif — flanking rules collapse the label to pure wayfinding. Chevron moves inside the label cluster.
D — Right chevron: Clears the left gutter so item names start at the same x-coordinate as in all other list contexts. Whisper-gray text, right-aligned control affordance.
